== Description ==

This little addon of <a href="https://wordpress.org/plugins/real-estate-manager/">Real Estate Manager</a> lets you easily filter property listings on map.

<h2>Shortcode</h2>

<ul>
<li>[rem_map_filters]</li>
</ul>

<h2>Additional Shortcodes</h2>
<ul>
<li>[rem_map_search_form]</li>
<li>[rem_map_area]</li>
<li>[rem_search_results]</li>
</ul>
